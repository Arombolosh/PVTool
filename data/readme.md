# Files needed during simulation

```
DB_Climate       - contains built-in climate data location files, shown in combo box
DB_PCMMaterials  - all m6 PCM Material files (complete), to be selected by user
project_template - directory structure to be copied into working directory on start of simulation
file_templates   - contains template files (d6p and insulation m6 file) that are read, modified and written
                   to working directory in each iteration
```
